# Installation

## MacOS & Linux
```sh
> git clone https://github.com/naufalardhani/uts_praktikum_pert-14
> cd uts_praktikum_pert-14 && pip3 install -r requirements.txt
> python3 main.py
```

## Windows
```sh
> git clone https://github.com/naufalardhani/uts_praktikum_pert-14
> cd uts_praktikum_pert-14 && pip install -r requirements.txt
> python main.py
```
